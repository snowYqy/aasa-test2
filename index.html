App Link Tester

输入一个 code 参数，点击按钮测试 Universal Link / Intent。

Open in iOS (Universal Link) Open in Android (Intent)

Tips:

  • iOS 使用 Universal Link，例如 https://你的域名/auth-callback?code=xxx。
  • Android 使用 Intent URL，格式 intent://...#Intent;scheme=yourapp;package=com.example.yourapp;end。
  • 替换下面 JS 中的 scheme 和 packageName 为你的真实配置。
